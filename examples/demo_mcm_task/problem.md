# Demo MCM Task: Urban Flood Response Prioritization

City Alpha faces repeated short-duration flooding after intense rainfall. The city has
limited emergency response resources and wants a transparent model for deciding which
districts should receive pumps, road crews, and temporary shelters first.

You are given a small district-level dataset with recent rainfall, drainage capacity,
population exposure, emergency call volume, critical facility count, road closure reports,
and historical flood loss. Build a modeling workflow that helps the city:

1. Construct a flood response priority score for each district.
2. Explain which indicators drive the priority score.
3. Recommend a robust resource allocation policy under limited resources.
4. Test whether the recommendations are sensitive to reasonable changes in indicator
   weights or data uncertainty.

Your final paper should include assumptions, a reproducible model, at least one vector
data figure, and a short discussion of limitations. If additional external data would
normally be useful, state how it would be sourced and how proxy variables can be used when
official data is unavailable.
